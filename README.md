> **Abstract**<br>
> This repository contains the **Flexbox Activity**, a foundational frontend college assignment. Its primary objective is to practice modern CSS layout techniques using the Flexible Box Module, ensuring clean, responsive, and adaptable UI components without relying on legacy floats or rigid positioning.

<br>



---

## <img src="https://icongr.am/feather/terminal.svg?size=24&color=A1A1A6" align="absmiddle" /> Execution & Setup

Because this is a static frontend project, no complex build steps, servers, or databases are required. To view the activity locally, simply use any modern web browser.

### 1. Clone & Execute
Navigate to your desired directory and clone the repository. You can open the `index.html` file directly in your browser.

```bash
# Clone the repository
git clone https://github.com/pedroforbeck/flexbox.git

# Navigate to the directory
cd flexbox

# Open the HTML file in your default browser (macOS)
open index.html

# For Linux: xdg-open index.html
# For Windows: start index.html
```
